У список основних знарядь стоматолога входять такі ручні стоматологічні інструменти: Стоматологічне дзеркало - діагностичний інструмент, який виконує дві функції: забезпечує візуальний доступ для дослідження в не проглядається ділянках ротової порожнини, а також використовується для виявлення хворобливості зубів. Стоматологічний пінцет. Призначений для роботи з іншими, дрібнішими інструментами, для транспортування ватних тампонів, перевірки рухливості зубів. Зонд - функціональний діагностичний інструмент. З його допомогою лікар проводить обстеження зубів на наявність пошкоджень, порожнин, досліджує стан дентину, перевіряє наявність больової реакції. Лоток для всіх перерахованих предметів. Крім основного набору ручного інструментарію, практично у всіх видах стоматологічного лікування використовується ротаційний інструмент для препарування та обробки твердих тканин (бори).

Компанія Future-Med працює на ринку стоматологічного обладнання вже понад 50 років. І за цей час досягла таких результатів - що справедливо вважається лідером в цій галузі. В багатьох звичних для сучасної індустрії речах вона виступала як новатор - даючи приклад для наслідування. Революційні погляди, є невідємною частиною філософії компанії - що, в поєднанні з високими технологіями та неперевершеним дизайном, створює унікальний образ Future-Med.
